# Air Quality Monitor & Predictor

This is a Streamlit application that monitors live air quality in India and predicts future AQI values using data from WAQI and OpenWeather APIs.

## Features

- **Live Monitoring:** View real-time air quality data on an interactive map.
- **Future Prediction:** Predict future AQI using a simple linear regression model.
- **CSV Export:** Export live data for all regions.
